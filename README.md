# Escape-Game-Combination
This is a web app that detects a series of keystrokes on the keyboard to deliver a clue or a solution for the rest of the escape game.  Compatible and designed for use with a Makey-Makey.

*Created for an unique event*

---Bind keys---
W,A,S,D,F,G,rightArrow

All keys are bindable in the "correspondance" table in "app.js" via usage of keyCode (https://keyCode.info).
